Aw21 Test.4.5.3 NMI 01

Does each synchronised or audio-only time-based media that is embedded via <object> or <embed> and that requires a text transcript pass one of the conditions below (except in special cases)?
  • A reference to the text transcript via a link or an anchor is available between <object> and </object>
  • A reference to the text transcript via a link or an anchor is available between <noembed> and </noembed>
NMI : No-check rule
