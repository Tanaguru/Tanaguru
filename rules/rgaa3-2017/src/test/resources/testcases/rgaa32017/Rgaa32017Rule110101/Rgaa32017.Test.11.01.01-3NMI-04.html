Rgaa32017 Test.11.01.1 NMI 04

Chaque champ de formulaire vérifie-t-il une de ces conditions ?
  • Le champ de formulaire possède un attribut title
  • Une étiquette (balise label) est associée au champ de formulaire
  • Le champ de formulaire possède une propriété aria-label
  • Le champ de formulaire possède une propriété aria-labelledby référençant un passage de texte identifié
  • The form field has a title attribute
  • A label (label tag) is associated with the form field

Passed: The select in the scope has a aria-label attribute not empty and an id. The other input are out of scope. By the order of reading (first aria-labelledby, then aria-label, then, if there is no id linked to a label, check the title) the aria-label is detected and checked event if there is the id. So, the aria-label needs to be checked if the content is pertinent or not.
